# SQL Practice Project

This repository contains my SQL learning journey and practice scripts.

## Project 1: Student Database Basics
In the file `01_sql_basics.sql`, I practiced the following:

* **Table Creation:** Used `CREATE TABLE` to define a schema for student data.
* **Data Entry:** Used `INSERT INTO` to add records into the table.
* **Basic Querying:** Used `SELECT *` to retrieve all information from the database.

### Concepts Covered:
1. Data Types (INT, VARCHAR)
2. Primary DDL (Data Definition Language)
3. Primary DML (Data Manipulation Language)
